# OVERVIEW
EN:
This project was created to practice the embedded system design workflow and PCB design process.
It is not a verified production-ready design, and the PCB has not been manufactured or physically tested.

The schematic is the primary reference document.

VI:
Dự án này được thực hiện nhằm luyện tập quy trình thiết kế hệ thống nhúng và kỹ năng vẽ PCB.
Đây không phải thiết kế hoàn chỉnh đã kiểm chứng thực tế, PCB chưa được sản xuất và thử nghiệm phần cứng.

Sơ đồ nguyên lý (Schematic) là tài liệu đáng tham khảo nhất trong repository.
